# nation-foreign-language-guide-zh

Paul Nation 的《What do you need to know to learn a foreign language?》（学语言你需要知道什么）简体中文翻译版。

---

## 关于本项目

本仓库提供了新西兰惠灵顿维多利亚大学 Paul Nation 教授关于外语学习指南的中文翻译。

- **原版阅读：** [What do you need to know to learn a foreign language? (PDF)](https://www.wgtn.ac.nz/lals/resources/paul-nations-resources/paul-nationspublications/publications/documents/foreign-language_1125.pdf)
- **翻译方法：** 关于本材料的翻译过程与思路，请参考我的博文：[如何翻译自用的英语书籍](https://ramen-curator.blogspot.com/2026/04/blog-post_29.html)。

## 仓库内容

本仓库目前包含以下两种格式的文件，供大家自由使用：

- **PDF 版本**：可直接阅读，适合多端预览。
- **DOCX 版本**：提供可编辑格式，方便后续的排版和校对。

## 为什么要开源（以及你可以如何帮忙）

本着 **“敏捷开发”** 的思路，为了让大家能迅速用上这份高质量的学习材料，我优先完成了内容的翻译并迅速发布了这一版。因为个人精力与时间较紧，且排版并非我所擅长，所以目前的内容在版式上还有提升空间。

**最终愿景：**
如果你有余力，非常欢迎下载 `.docx` 文件进行精美的排版设计。如果在大家的努力下，我们能得出一份排版精良的最终版本，就可以直接与 Paul Nation 教授交涉，争取将这份中文版挂载到他的官方网站上。

当这份翻译被官方正式收录之时，也就是本仓库光荣归档之日。欢迎大家提交 Pull Request！
